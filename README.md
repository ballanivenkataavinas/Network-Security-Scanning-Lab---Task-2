# Network Security & Scanning Lab

# Overview
This repository contains my hands-on work for **Task 2 – Network Security & Scanning**.  
The objective is to learn reconnaissance, network scanning, vulnerability assessment, and packet analysis using industry tools.

# Objectives
- Perform reconnaissance (passive & active)
- Conduct network scanning using Nmap
- Identify services and operating systems
- Perform vulnerability scanning
- Analyze network traffic using Wireshark
- Understand firewall basics

# Tools Used
- Kali Linux
- Metasploitable2
- Nmap
- Wireshark
- Burp Suite
- Netcat

# Key Activities
- Information gathering using Whois & Nslookup  
- Network scanning and port analysis  
- Service enumeration and OS detection  
- Vulnerability assessment using scanning tools  
- Packet capture and traffic analysis  
- Basic firewall rule configuration  


# Deliverables
- Screenshots of scans and analysis  
- Scan reports (Nmap & vulnerability tools)  
- Detailed notes for each day  
- 5-minute demo video explaining findings  

#  Outcome
This lab helped me build a strong foundation in:
- Network reconnaissance  
- Scanning and enumeration  
- Traffic analysis  
- Cybersecurity tools usage
